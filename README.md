# Python Learning Repository

This repository contains a collection of Python scripts and exercises completed as part of a daily learning challenge. Each file represents work done on a specific day, covering various Python concepts from basic data types to algorithms.

## Files Overview

- **day2.py**: Basic input handling and number operations
- **day3.py**: Introduction to lists, indexing, and list operations
- **day4.py**: Working with tuples, sets, dictionaries, and type casting
- **day5.py**: Type conversion, operators, and conditional statements
- **day6.py**: String methods and conditional logic with quiz example
- **day7.py**: Similar to day6, focusing on conditional statements
- **day8.py**: Introduction to loops with a simple printing loop
- **day9.py**: If-else conditions and positive/negative number checking
- **day10.py**: Implementation of Radix Sort algorithm
- **day11.py**: Grading system using elif statements
- **day12.py**: Loops and pattern printing
- **day14.py**: Introduction to functions and basic function examples
- **day15.py**: File handling in Python (reading, writing, modes, error handling)

## Requirements

- Python 3.x

## How to Run

Each Python file can be run individually using:

```bash
python dayX.py
```

Replace `X` with the specific day number.

## Learning Objectives

This repository demonstrates:
- Basic Python syntax and data structures
- Input/output operations
- List, tuple, set, and dictionary manipulations
- Algorithm implementations (e.g., sorting algorithms)
- Problem-solving through coding exercises
